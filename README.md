Libraries Used

<stdio.h> – Basic input/output functions (printf, scanf).

<conio.h> – Console functions like getch() and kbhit() (Windows/Turbo C only).

<windows.h> – Windows API functions for delays, colors, and console control (Sleep).

<stdlib.h> – Utility functions like rand, srand, and system.

<time.h> – Time functions, mainly for seeding random numbers (time(0)).
